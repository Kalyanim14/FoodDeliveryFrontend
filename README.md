# Food Delivery Application - Frontend

A full-stack food delivery web application built with React and integrated with a Spring Boot backend. The platform enables users to discover restaurants, browse menus, manage carts, place orders, and securely access personalized features through authentication and authorization.

## Features

* User Registration and Login
* JWT-Based Authentication
* Browse Restaurants and Food Menus
* Add, Update, and Remove Cart Items
* Place Orders and View Order History
* Responsive User Interface
* Admin Dashboard for Restaurant and Menu Management
* RESTful API Integration with Spring Boot Backend

## Getting Started

### Prerequisites

* Node.js (v18+ recommended)
* npm

### Installation

Clone the repository:

```bash
git clone https://github.com/Kalyanim14/FoodDeliveryFrontend.git
```

Navigate to the project directory:

```bash
cd FoodDeliveryAppFrontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

## Backend Repository
